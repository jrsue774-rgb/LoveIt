Guia completa para aprender Linux desde cero usando la terminal Bash.

<!--more-->

## 1 Que es Linux

Linux es un sistema operativo de codigo abierto creado en 1991 por Linus Torvalds. Es usado en servidores, telefonos Android y supercomputadoras de todo el mundo.

## 2 Por que aprender Linux

- Es completamente gratuito
- Muy demandado en el mundo laboral
- Te ensena como funciona realmente una computadora
- La terminal te da control total del sistema

## 3 Primeros pasos

Si usas Windows puedes empezar con Git Bash para practicar sin instalar nada mas.

### 3.1 Instalar Git Bash

Descarga Git desde https://git-scm.com e instalalo en Windows. Incluye Git Bash automaticamente.

### 3.2 Abrir la terminal

Busca Git Bash en el menu inicio y ejecutalo. Veras una pantalla negra con un cursor, eso es la terminal.

## 4 Comandos basicos

```
bash
pwd       # ver en que carpeta estas
ls        # listar archivos
cd        # entrar a una carpeta
mkdir     # crear una carpeta
clear     # limpiar la pantalla
```

## 5 Consejos para principiantes

- Practica los comandos todos los dias
- No tengas miedo de cometer errores
- La practica hace al maestro
