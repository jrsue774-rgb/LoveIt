Guia paso a paso para instalar Linux en tu computadora desde cero, sin experiencia previa.

<!--more-->

## 1 Que necesitas

- Una computadora con al menos 4GB de RAM
- Un USB de minimo 8GB
- Conexion a internet
- Tiempo y paciencia

## 2 Elegir una distribucion

- Ubuntu — la mas popular y facil de usar
- Linux Mint — muy similar a Windows, ideal para migrar
- Debian — estable y confiable

## 3 Descargar la imagen ISO

Ve al sitio oficial de la distribucion que elegiste y descarga el archivo ISO.

## 4 Crear el USB booteable

1. Descarga Rufus en Windows
2. Conecta el USB
3. Abre Rufus y selecciona el USB
4. Selecciona el archivo ISO
5. Haz clic en Empezar

## 5 Instalar Linux

1. Reinicia la computadora con el USB conectado
2. Entra al menu de arranque con F12 o F2
3. Selecciona arrancar desde USB
4. Sigue el asistente de instalacion
5. Reinicia cuando termine

## 6 Primeros pasos despues de instalar

Actualiza el sistema desde la terminal con estos comandos.
